# Тестовый проект на 1С:Enterprise 8.3

![1C:Enterprise Logo](https://obrazovanie.1c.ru/upload/medialibrary/ad5/cevdpx59ghzm8ffi43oe0ogcc4idxvtc.png)

Это учебный проект для освоения основ программирования в 1С:Enterprise 8.3. Проект содержит примеры работы с основными объектами платформы и базовые конструкции языка.

## Содержание проекта

### 1. Основные объекты платформы
- **Справочники**: Примеры работы с иерархическими справочниками
- **Документы**: Реализация простых документов с проведением
- **Отчеты**: Примеры построения отчетов с использованием СКД
- **Обработки**: Интерактивные обработки данных
- **Регистры сведений**: Хранение и получение данных

### 2. Основы программирования
- **Переменные и типы данных**
- **Условные операторы**
- **Циклы**
- **Процедуры и функции**
- **Работа с датами и строками**
- **Обработка исключений**

## Требования
- 1С:Enterprise 8.3 (версия не ниже 8.3.10)
- Git клиент (для установки из репозитория)


## Установка
1. Создайте новую информационную базу
2. Загрузите файлы конфигурации (*.cf или *.cfe)
3. Обновите конфигурацию базы данных

## Установка из GitHub репозитория

### Способ 1: Через HTTPS
```bash
git clone https://github.com/DenisKhudyakov/1C_Demo.git
cd 1C_Demo
```
### Способ 2: Через SSH
```bash
git clone git@github.com:DenisKhudyakov/1C_Demo.git
cd 1C_Demo
```
